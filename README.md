# OpenGLVSCode
Using OpenGL in C++ on VSCode.

## Goal
1. Successfully build C++ on VSCode using OpenGL
1.1. Setup VSCode to use C++
1.2. Add OpenGL dependencies

## Setup
### Current environment.
Windows 10 with WSL

```
sudo apt-get update
sudo apt-get install build-essential libglfw3-dev libglew-dev libglm-dev
```
